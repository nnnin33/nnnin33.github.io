自动控制原理学习

课件地址：https://github.com/nnnin33/ACS_THU

自动控制原理-大观

课程整本书的内容逻辑其实就是：建模 –分析–设计

内容逻辑

系统建模是第2章：

  1. 先是讲了两种描述物理系统的方法：微分方程和传递函数。传递函数也是由微分方程进行拉普拉斯变换得到的。

    微分方程是属于时域的语言，而传递函数是复频域(s-Domain, s)的语言，把微积分运算变成乘除法了。

    传递函数的定义是：输出的拉普拉斯变换 除以 输入的拉普拉斯变换，而且初始条件必须为0。

  2. 然后还讲了一个方框图。

    术语：前向通路、开环、闭环。

    计算：利用梅森增益公式 (Mason's Rule) 或者方框图变换，把一堆框化简成一个传递函数。

系统分析是3-5章：

  1. 第3章是时域分析。是直接解方程，看输出随时间 t 怎么变。

    但是这里面的这个方程 ，其实是先解传递函数的方程C(s)=G(s)×R(s)，然后再 逆拉普拉斯变换 把算出来的 C(s) 变回 c(t)，这时候我们就能看到输出随时间 t 怎么变了。

    主要研究的东西有3个：稳定性、静态误差（也叫稳态误差）、动态响应（也叫动态性能）

    • 稳定性：研究系统能正常工作吗？会不会发散？

      判断：线性定常系统稳定 当且仅当 其特征方程的所有根具有负实部；线性定常系统稳定 当且仅当 其传递函数的所有极点位于复平面的左半平面；劳斯表``第一列系数全为正数

      计算：劳斯判据（不用解方程就能判断稳不稳定）

    • 稳态误差：看最后输出结果准不准？

      计算：终值定理，以及判断系统的型别（Type 0, 1, 2…）

    • 动态性能：看系统的反应快不快？震荡大不大？

      术语：超调量 、调节时间 、上升时间 等等

      关联：这些指标主要由极点 (P) 的位置决定。

  2. 第4章是频域分析。是给系统输入正弦波，看输出正弦波的幅度和相位变了多少

    术语：伯德图 (Bode Plot)、奈奎斯特图 (Nyquist Plot)。

    稳定性判据：奈奎斯特判据（通过开环特性判断闭环稳定性）。

    性能指标：相角裕度 (Phase Margin)、幅值裕度。裕度越大，系统越稳。

  3. 第5章是根轨迹法。研究当系统某个参数（增益 K）变化时，系统的闭环极点在复平面上怎么跑？

    这一章直接把稳定性、动态性能这些参数性能都包括进来了。

    计算：渐近线、分离点等等，画轨迹。

系统设计是第6章：

  1. 这一章是在原系统中串联或并联一个新的环节：控制器 Gc

    校正方法：PID(比例-积分-微分)、超前校正、滞后校正

    计算：利用根轨迹法或伯德图计算控制器Gc的表达式。

其他特殊情况在7-8章：

  1. 第七章讲非线性系统。不能用叠加原理的。一般是现实情况中的死区、饱和、继电器特性之类的问题。

    采用的解决方法是相平面法和描述函数法，描述函数法类似于谐波分析

  2. 第八章讲采样控制系统。想要解决的是当电脑控制是断续的数字信号，而不是连续的模拟信号的时候遇到的问题。

    采用的方法是用z变换把 s 域搬到 z 域，用jury朱利判据

核心关联

极点 (P) 决定了很多东西：

  • 极点在左半平面 → 稳定。
  • 极点离虚轴越远 → 反应越快。

时域分析直接看极点。

根轨迹看极点怎么动。

频域分析通过开环频率特性间接推断闭环极点的行为。

解释：

时域分析传递函数的分母（特征方程）解出来的根，就是极点 s=σ+jω。

c(t) 的解里包含 e^st 项。

  • 实部 σ：如果是负数 (e^−at)，系统最后会稳定；如果是正数 (e^at)，系统会不稳定。
  • 虚部 ω：如果有虚部 (e^jωt=cosωt+j sinωt )，系统就会震荡

根轨迹其实就是看增益 K （放大器的倍数） 从 0 变到无穷大时，特征方程 1+KG(s)H(s)=0 的根（极点）会不停地变，然后根据渐近线、分离点等等，画出极点移动的轨迹。

比如可以看到当 K 调到 10 的时候，极点刚好跑到左半平面，系统就是稳定了；K 再大一点，极点虚部变大，震荡太厉害了，就不是我们想要的了。

时域分析是1、2阶的简单系统，频域分析是高阶复杂系统。

频域分析是不知道系统的传递函数的时候，通过开环来推断闭环。

目标还是要找闭环极点，看他是不是在左半平面，是不是稳定的。但是由于解不出来闭环极点，所以转而去画开环传递函数 G(jω)H(jω) 的图（伯德图或奈奎斯特图）。这很容易，给个正弦波测一下就出来了。然后再根据奈奎斯特判据就可以推断闭环极点在哪里

由于有定理：柯西辐角原理：闭环特征方程是 1+G(s)H(s)=0。所以当 G(s)H(s) 的值等于 -1 (即 −1+j0) 时，系统就到达了临界点。所以只需要看开环频率特性的曲线，有没有包围复平面上的点 (-1, j0)

  • 如果包围了，说明闭环极点跑到了右半平面 → 不稳定。

  • 如果不包围且满足开环系统本身稳定，说明闭环极点都在左边 → 稳定。

开环系统本身稳定的要求是：环传递函数 G(s)H(s) 在右半平面没有极点（P=0）

Z=P−N

  • Z (Zero)：闭环系统在右半平面的极点数。（目标是Z=0，这样系统才稳定）。

  • P (Pole)：开环系统在右半平面的极点数。（这是已知的，看一眼开环传递函数的分母就知道了）。

  • N (Number)：开环频率特性曲线 G(jω)H(jω) 逆时针包围 (-1, j0) 点的圈数。如果是顺时针包围，就是负数。

    不包围不一定稳定，稳定前提是P=0

传递函数

线性定常系统的传递函数零初始条件下，系统输出量的拉氏变换与输入量的拉氏变换之比。

开环：$G(s)H(s)$

闭环：$G(s)/1+G(s)H(s)$

这里的 s ：

  • 如果输入是直流（常数 A），此时 s→0。最终输出就是 G(0)×A。

  • 如果输入是正弦波 (sinωt)，此时 s→jω。涉及频域分析里面的频率特性

要注意的点：

  1. 零初始条件

  2. 系统的固有特性，与输入无关

  3. 传递函数 = 单位脉冲响应 k(t) 的拉普拉斯变换。

  4. 仅限线性定常系统

题型3：判断实际稳定性

假设一个单位负反馈系统，前向通路是 $G(s)=10/s(s+1)$，判断这个系统实际工作时是否稳定，我们要看哪个方程的根？

(A) $s(s+1)=0$ (B) $s(s+1)+10=0$

答案是B，稳定性必须要看闭环极点。

而A的开环极点的作用是：因为求B的根有的时候会遇到解高阶方程的情况，会很困难。所以通过分析A的极点来推测B。

  • 根轨迹起点。当增益 K=0 时，系统还没开始控制，闭环极点就等于开环极点
  • 奈奎斯特判据。$Z=P−N$，$ P$ 就是 A 的根在右半平面的个数。必须先看 A 稳不稳定，才能决定数圈圈时要数几圈（$N$ 应该是多少），最终判断出 B 稳不稳定（$Z$ 是否为 0）
  • 决定系统型别。开环传递函数的分母上 s=0 的极点个数：$G(s)H(s)=K/s^ν(τs+1)…$
    • v=0个：0型。没有积分环节，跟不上阶跃输入，会有稳态误差
    • v=1个：1型。有一个原点极点。可以跟的上阶跃输入，误差为0；但是跟不上斜坡输入
    • v=2个：2型。有两个原点极点，跟得上阶跃输入和斜坡输入。
  • 决定伯德图的转折点。开环极点 $p=−10$ (即对应环节 $1/0.1s+1$) 在频率 $ω=10$ (即 $1/T$ ) 的地方，幅频特性曲线就会向下拐弯，斜率减小 $20dB/dec$
  • 设计控制器时零极点对消。开环系统里的极点离虚轴太近（导致反应慢），或不稳定。比如$G(s)=1/(s+0.1)(s+5)$里面$s=−0.1$ 这个极点太靠近原点，导致系统反应超级慢，可以设计一个控制器$G_c(s)=s+0.1/s+10$，相乘之后，$(s+0.1)$分子分母抵消了，系统变得像 $(s+10)$ 那样快
