纯小白向–Jetson orin nano super新机配置

date: 2025-12-21

这是我的第一篇博客，内容就是我考虑到的拿到一块新板子以后，装好jetpack进去以后还要配置的一些东西。

主要是买的板子厂家已经直接把系统都全部配置好了，我看了一下配置什么也都挺新的，所以就没有烧录安装sdk的环节了。想到之前大一的时候拿到的别人用过的乱糟糟的jetson nano的旧板子，拿到看到里面文件系统那么乱就觉得脏脏的不想用了，自己重新装就又搞了好久，那个时候才刚接触计算机而且又没有gpt，每次遇到bug在互联网上查别人的攻略都要查好久还是解决不了。现在真是忆苦思甜上了hhhh

检查Jetpack版本：

R36.x.x就是最新的jetpack6版本了，一般都是ubuntu22.04 。一般jetson都是装ubuntu系统的，要不然卡会驱动不起来。

确认系统挂载情况，是不是真的安装在自己的硬盘上面的：

看size是不是差不多你的硬盘的大小来确定。

查看CUDA装好没：

看release后面 x.x 是多少。我的就是12.6

jetson orin nano super最高是可以达到40TOPS/INT8的性能，TOPS=万亿次操作每秒，INT8是8位整数精度，通常用于推理，也就是模型已经训练好了拿来用的时候。速度更快，内存占用更小。相比于老的jetson nano只有0.5TOPS/FP16。

还有一个注意事项说就是板子是arm架构的（aarch64），不要下载写amd64或者x86_64的安装包，要找aarch64或者arm64版本的。如果在网络还没有配置好的时候想要安装什么软件最好是先找个u盘下载好了以后拷过来在板子上安装，速度快得多。

接线

插上电源线以后会亮黄灯。（我看到的反正是黄的，拍照出来竟然是绿色的）板子上最多是四根线：

- 电源线
- DP-hdmi线（一般会要用到一个DP公转HDMI母的转接头，DP口是缺一个角，HDMI口是缺两个角的梯形）
- 鼠标
- 键盘

要注意的是触摸板（显示屏）电源线是那个圆头头的不是type-c头，有专门的电源线。我第一次找了根type-c和usb的线接到板子上了，还以为是板子给触摸板供电。蠢蠢蠢死了

网络

我先在自己的电脑上下载了clash verge安装包以后拷到板子上安装的，下载地址：clash_verge_rev

拖拽到左侧栏的 Home在这个文件夹下打开terminal

导入了订阅 但是还是链接不上外网怎么办：

  1. clash verge – setting – system proxy打开

  2. 看节点挂没挂

  3. ubuntu setting – network – network proxy – manual – http/https/socks host都填一致的clash verge里面显示的端口号（新版一般是7897）

  4. 检查端口到底有没有对：

    出现 * Connected to 127.0.0.1 port 7897，然后是一堆 TLS 握手信息，最后输出了网页代码，说明clash是好的，是浏览器的问题。

  5. firefox setting – proxy – 把use system proxy settings改成 manual proxy configuration，http proxy填127.0.0.1，端口填7897，然后勾选also use this proxy for https

我就是到这一步解决的
